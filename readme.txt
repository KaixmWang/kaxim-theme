一套适用于 WordPress 的简洁、轻量的极简博客主题。支持三套主题模式（默认/护眼/夜晚）、页面加载动画、滚动淡入动画，专注阅读体验。

演示：https://kaxim.wang/blog

== Description ==

Kaxim Theme 完全由AI创建，不含一丝人工代码的 WordPress 极简博客主题。设计理念是"全都删掉"——通过极简的界面、舒适的阅读体验，让写作和阅读回归本质。

**主要特性：**

* 三套主题模式：默认（白底黑字）、护眼（暖色调）、夜晚（深色背景）
* 页面加载淡入动画，滚动渐显效果
* 单栏居中布局，专注内容阅读
* 响应式设计，适配桌面和移动端
* 支持自定义 Logo、社交链接、页脚信息
* 内置中文阅读时间估算
* 性能优化：无外部依赖、CSS 变量驱动、图片懒加载
* 无障碍支持：跳过链接、键盘导航、语义化 HTML
* 遵循 WordPress 官方编码标准和安全规范

== Installation ==

1. 下载主题 ZIP 文件
2. 进入 WordPress 后台 → 外观 → 主题 → 添加新主题 → 上传主题
3. 上传 ZIP 文件并启用
4. 进入 外观 → 自定义 配置主题选项

== Frequently Asked Questions ==

= 如何切换主题模式？ =

在页面底部找到"主题"切换器，点击"默认"、"护眼"或"夜晚"即可切换。设置会保存在浏览器中。
